# PEI Foodie Road Trip 2026

A self-contained, interactive family road-trip itinerary for a Vaughan, Ontario to Prince Edward Island trip, August 7–14, 2026.

## Open the itinerary

Open [index.html](index.html) in a modern web browser. The dashboard works as a single local file; external map, photo, and source links require an internet connection.

## Contents

- `index.html` — interactive itinerary dashboard
- `roadtrip-html-master-prompt.txt` — original planning and generation brief

## Before traveling

Confirm operating hours, reservations, road conditions, fuel availability, and weather before departure. This repository contains personal travel plans and is intended for the trip participants.
